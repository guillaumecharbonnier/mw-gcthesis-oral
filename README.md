# mw-gcthesis-oral
Metaworkflow plugin to compile my thesis oral presentation

## Requirements:
- conda
- snakemake

## Instructions:
```
git clone git@github.com:guillaumecharbonnier/mw-lib.git
git clone git@github.com:guillaumecharbonnier/mw-gcthesis-oral.git
cd mw-gcthesis-oral
./compile-slides.sh
```
